## Description
This is an open-source Q-SYS plugin for controlling and monitoring [Generation 2](https://support.luminex.be/portal/en/kb/articles/gigacore-generation-2-devices) [Luminex](https://www.luminex.be/) GigaCore switches from [Q-SYS](https://www.qsys.com/).
